树莓派迅雷远程下载

2019-03-09
#野生技术协会 #树莓派

生命不止，折腾不息。树莓派常被创客们（比如我）折腾，捣鼓出一些有意思的东西。拥有一台能24小时运行还低能耗的下载姬绝对爽。

安装迅雷

绑定

先初始化

sudo ./portal

然后会有一大段输出信息，找到激活码

THE ACTIVE CODE IS: n***w

到这里绑定

绑定之后可以通过访问 http://192.168.1.37:9000/getsysinfo 来查看设备信息，其中192.168.1.37换成你的树莓派的 IP 地址，可以看到类似如下的信息

[ 0, 1, 1, 1, "", 1, "2.215.3.310", "15*******86", 1, "6*******8", 0 ]

其中，
第一项：0表示返回结果成功
第二项：1表示网络正常，0表示网络异常
第四项：1表示设备已绑定，0表示设备未绑定
第五项：未绑定的情况下，为绑定的需要的激活码
第六项：1表示磁盘挂载成功，0表示磁盘挂载失败
第八项：绑定的迅雷账号

启动，停止和开机启动迅雷服务

启动迅雷，命令行执行：

sudo /home/pi/thunder/portal

停止迅雷，命令行执行：

sudo /home/pi/thunder/portal -s

开机启动迅雷服务：

sudo nano /etc/rc.local

在exit 0前一行，写上挂载命令，即：

/home/pi/thunder/portal

用树莓派做FM发射器架设小型电台

2019-02-22
#野生技术协会 #树莓派

树莓派的GPIO引脚，可以作为时钟信号输出，于是可以把音频信号通过树莓派进行FM调制后从该引脚发出，这样树莓派就成了一台微型FM电台，可以自己指定频率，打开调频收音机，调到对应频道就能收到树莓派播放的微型 FM 广播。

本片教程通过安装fm_transmitter来将树莓派变成微型FM电台。
安装编译工具 

sudo apt-get install make gcc g++

下载fm_transmitter 

sudo git clone https://github.com/markondej/fm_transmitter

然后进入 fm_transmitter 目录，执行make： 

cd fm_transmitter
sudo make

树莓派FM发射器开启命令格式为： 

sudo ./fm_transmitter [-f frequency] [-r] filename

其中，frequency为指定的频道，filename为要播放wav格式的音频文件。

例如，使用以下命令在100.6频道播放在本地的star_wars.wav(文件音频必须是wav文件16bit 44.1kHz单声道)：

sudo ./fm_transmitter -f 100.6 -r ./star_wars.wav

没有在树莓派GPIO 4（PIN7）上接一根长长的杜邦线增强信号，直接播放，使用手机收听，在1米内音质很清晰，3米的范围仍然可以听的见，超过3米就不行了。
