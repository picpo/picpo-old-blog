树莓派迅雷远程下载

发表于 2019-03-09

生命不止，折腾不息。树莓派常被创客们（比如我）折腾，捣鼓出一些有意思的东西。拥有一台能24小时运行还低能耗的下载姬绝对爽。

安装迅雷

绑定

先初始化

sudo ./portal

然后会有一大段输出信息，找到激活码

THE ACTIVE CODE IS: n***w

到这里绑定

绑定之后可以通过访问 http://192.168.1.37:9000/getsysinfo 来查看设备信息，其中192.168.1.37换成你的树莓派的 IP 地址，可以看到类似如下的信息

[ 0, 1, 1, 1, "", 1, "2.215.3.310", "15*******86", 1, "6*******8", 0 ]

其中，
第一项：0表示返回结果成功
第二项：1表示网络正常，0表示网络异常
第四项：1表示设备已绑定，0表示设备未绑定
第五项：未绑定的情况下，为绑定的需要的激活码
第六项：1表示磁盘挂载成功，0表示磁盘挂载失败
第八项：绑定的迅雷账号

启动，停止和开机启动迅雷服务

启动迅雷，命令行执行：

sudo /home/pi/thunder/portal

停止迅雷，命令行执行：

sudo /home/pi/thunder/portal -s

开机启动迅雷服务：

sudo nano /etc/rc.local

在exit 0前一行，写上挂载命令，即：

/home/pi/thunder/portal
